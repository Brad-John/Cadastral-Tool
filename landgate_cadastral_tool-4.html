06 · Survey Marks (SSM / BM)
⚠ GDA94-referenced output selected. SSM coordinates will be read from Landgate's published GDA94 attribute fields where available — however not all marks have these fields populated. Marks missing GDA94 attributes will fall back to a 7-parameter datum transformation (±50–140mm accuracy). For best results use PCG2020 or MGA2020, which use SLIP's native GDA2020 coordinates with no datum shift.
